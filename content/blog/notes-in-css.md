date: 2021-08-28

Notes can be used to notify the user about something special: danger, success, information or warning

## Step 1: Add HTML

### Example

```html
<div class="danger">
  <p><strong>Danger!</strong> Some text...</p>
</div>
```

## Step 2: Add CSS

### Example

```css
.danger {
  background-color: #ffdddd;
  border-left: 6px solid #f44336;
}
```
